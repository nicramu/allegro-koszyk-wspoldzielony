Dziel się przedmiotami w koszyku Allegro ze znajomymi. Eksportuj swój koszyk jako "id", które możesz podesłać znajomym. Jednym kliknięciem dodaj wybrane przedmioty z koszyka pod swoje id, a znajomi od razu będą je widzieć. To alternatywa dla usuniętej funkcji prośby o zakupy. Dane są przetrzymywane w chmurze, ale w żaden sposób nie identyfikują konkretnych osób, jedyne co przechowuję to wygenerowane id (które nie ma nic wspólnego z kontem Allegro) i zawartość koszyka.
